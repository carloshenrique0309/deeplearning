# Notebooks

Use esta pasta para os experimentos principais do projeto.

Sugestao de ordem:

- `01_exploracao_dados.ipynb`: entender quantidade de reviews, classes e exemplos.
- `02_preprocessamento_modelo.ipynb`: tokenizacao, vetorizacao e treinamento.
- `03_avaliacao_testes.ipynb`: metricas, matriz de confusao e previsoes novas.

As funcoes auxiliares simples podem ficar em `src/imdb_data.py`.
